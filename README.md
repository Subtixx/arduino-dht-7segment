
## 

1. 7-Segment Display
2. DHT22

4. 7 Resistors
5. About 22 Cables (10 for the display, 7 for the resistors, 2 push-button, 3 DHT22)

## Display

I am using an common anode 7-segment display with 3 digits (2 = Full 7-Segment, 1 = 3-Segment).
